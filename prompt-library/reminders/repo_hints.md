Repository Hints Reminders

Repo Roles
- Respect writable versus reference-only repo boundaries.
- Never commit from reference-only repositories.

Repo Profiles
- Use profile-declared commands, protected paths, and dangerous operations.
- Escalate caution around operator-sensitive behaviors.

Compatibility
- Preserve public contracts unless change is explicitly required.
- Flag cross-repo impact when touching shared APIs, schemas, or build surfaces.
