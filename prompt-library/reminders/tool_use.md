Tool Use Reminders

Execution Discipline
- Prefer dedicated tools over shell when available.
- Use parallel read-only calls when gathering context.
- Keep tool calls scoped to the current step objective.

Safety
- Quote paths and arguments robustly.
- Avoid destructive commands unless explicitly requested.
- Never bypass safety constraints to work around denied permissions.

Observability
- Capture command results that support conclusions.
- Record errors verbatim when they drive decisions.
- Avoid noisy output when a concise excerpt proves the point.
